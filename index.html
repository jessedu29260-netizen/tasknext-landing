RULES

Use clamp() for fluid sizing: clamp(2rem, 5vw, 4rem) for hero headings.

Line-height: 1.5 for body, 1.1–1.2 for large headings.

Max content width: 1280px with auto margins.

Letter-spacing: -0.02em on headings, 0 on body.

Never use more than 3 font sizes on a single screen.

British English throughout all user-facing text.

Logo Usage Rules

Clear Space

Minimum clear space around the logo equals the height of the "T" in TaskNext. Never crowd the logo.

Minimum Size

Digital: 32px height minimum. Print: 15mm minimum. Below this, use the icon-only mark.

Don'ts

Never rotate. Never stretch. Never place on busy backgrounds without a glass container. Never change the gradient colours.

Section 06

Visual Identity Guidelines

Colour usage, motion principles, and storytelling voice rules.

Full Colour Usage Rules

#050507 Void: Always the background. Never pure black (#000). The extra warmth creates depth.

#818CF8 Indigo: Primary action colour. Buttons, links, active nav states, progress bars. Use at full opacity for CTAs, 20% opacity for subtle tints.

#C084FC Orchid: Accent-only. Organism glows, Ory's antenna tips, hover states on premium elements. Never use for primary text or large backgrounds.

Indigo→Orchid Gradient: Reserved for Ory-specific elements: the Ory logo, IQ score numbers, premium badges, and the ".AI" in the wordmark. Never apply to generic UI elements.

White at 50%: Secondary text, descriptions, timestamps. White at 30%: Tertiary (hints, watermarks). White at 100%: Headlines and primary content only.

Glass surfaces: Always rgba(255,255,255,0.03) fill with rgba(255,255,255,0.08) border and 40px backdrop-blur.

Motion Principles

Breathing Pulses

4-second ease-in-out scale (1.0→1.06→1.0) on all organism elements. This is the heartbeat of the brand. Applied to: Ory icon, IQ gauge, agent nodes.

Thinking Shimmers

Horizontal shimmer effect during AI generation. Use on loading bars, content preview placeholders, and the "Ory is thinking..." state.

Celebratory Effects

Soft confetti burst on milestones (IQ score increase, first post scheduled, founding member badge). Always subtle and brief (1.5 seconds max). Never loud or distracting.

Storytelling Voice in Copy

DO

✓ Use "Ory" as a character — it speaks, acts, and has personality.

✓ Tell stories: "Once upon a time...", "Plot twist:", "Week 1 → Week 8"

✓ Be warm and encouraging: "You're crushing it!", "Keep feeding Ory!"

✓ Use the organism metaphor naturally: breathes, learns, heals, grows.

✓ Make the user the hero: Ory is the sidekick, not the star.

✓ Use British English throughout.

DON'T

✗ Use corporate language: "leverage", "synergy", "streamline".

✗ Be robotic or cold: "Process complete. Action required."

✗ Over-promise: "100% success rate", "guaranteed results".

✗ Use more than one exclamation mark per message.

✗ Make Ory sound desperate or salesy.

✗ Forget the fun: every interaction should have a tiny spark of warmth.
